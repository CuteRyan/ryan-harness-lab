## 1. 4스킬 책임 분리 — 시간축 비유

```
        과거 ←──────────── 현재 ────────────→ 미래
                              │
   /history       /handoff   /checklist   /todo
   (완료 기록)   (세션 인계)  (지금 작업)  (백로그)
        │            │           │           │
   docs/history/  HANDOFF.md  .checklist.md  .todo.md
```

| 스킬 | 시제 | 핵심 파일 | 수명 |
|---|---|---|---|
| `/todo` | 미래 | `.todo.md` | 프로젝트 전체 (상시) |
| `/checklist` | 현재 | `.checklist.md` | 단일 작업 사이클 |
| `/handoff` | 미완 (세션→세션) | `HANDOFF.md` | 다음 세션 시작 시 소멸 |
| `/history` | 과거 | `docs/history/YYYY-MM-DD.md` | 영구 |

근거 강도: **강** (01_todo_spec.md L67~73 표 + 02_handoff_spec.md L77~84 표 직접 인용)
