## 2. AM — A2 마무리 (영어 prompt 검증 미달)

### NativeCommandError fix (인프라 블로커 해소)
- **발견**: G1 1차(2026-04-27) 실패 추적 — Codex/Gemini retry 모두 catch 발동
- **진짜 원인**: PS 5.1 NativeCommandError + `$ErrorActionPreference='Stop'` 조합. CLI stderr 정보 메시지가 RemoteException으로 wrap → terminating error 승격 → try 블록 catch 추락
- **재현 매트릭스 실측**:
  - 부모 세션 직접 호출: ✅ 정상 (7.4s, PONG)
  - Start-Job + EAP 미설정: ✅ 정상 (6.7s, PONG)
  - Start-Job + EAP='Stop' + try/catch: ❌ catch 발동 (FullyQualifiedErrorId: NativeCommandError)
- **수정**: `run-codex.ps1` + `run-gemini.ps1` 의 native CLI 호출 직전·직후만 EAP를 'Continue'로 격리. `$LASTEXITCODE` 검사는 EAP 무영향이라 그대로 작동
- 검증: Start-Job + EAP='Stop' + try/catch 재현 테스트 → catch 발동 안 함, "TRY-OK exit=0 outputLen=4 PONG"

## 5. B 방식 2단계 — 메인 Claude 5게이트 (5 항목 A~E)

### SKILL.md Step 3 5게이트 박음
1. **라인 실측 검증** — 환각 차단. 인용 라인 ≥ 원본 라인 수면 자동 [반박] 환각
2. **반박/유보 최소 1건** — 모두 [반영]이면 자기 검열 의무
3. **근거 강도 표시** — 강(코드 인용) / 중(원리) / 약(추상). 약 ≥ 50%면 헤더 경고
4. **통계 표 강제** — 첫 블록에 전체 N / [반영] N / [유보] N / [반박] N / [환각] N
5. **자기 비판 한 줄** — 마지막 블록에 "이번 종합에서 놓쳤을 가능성 1가지" 강제. "없음" 답변 금지

### E 검증 — H 결과로 종합 보고 작성
파일: `docs/feedback/2026-04-28_claude_g3_sample_20260428-161902-종합.md`

5게이트 발현 자체 검증 결과 **이전 1차 보고에서 빠뜨렸던 것 2가지가 자동 노출**:
1. **Gemini 라인 정밀도**: 5건 중 4건이 ±1~2 인접 라인 부정확. 환각은 아니지만 정밀도 약점. 1차 보고에선 매크로 비율(75%)만 봤지 정밀도까진 안 짚었음
2. **시스템 설계 차원 사각지대**: 동시성·트랜잭션·테스트 격리·로깅 부재. 3 CLI 모두 능동 탐색 안 함. 게이트 5가 없었으면 이번에도 통과시켰을 것

→ **게이트가 실제로 sycophancy 회귀를 막아준다**는 게 1회 실측으로 확인.

## 6. 의사결정

- **A2 영어 전환 롤백**: 효과 미미(38% → 40%) + 부작용(Claude Sub G2' INVALID 회귀). Why: 모델 본성을 prompt 언어로 못 풀음
- **B 방식 채택**: 프롬프트 파일화의 진짜 가치 = 인코딩 위험 0 + 형식 준수 100% (실측 입증). Why: PowerShell argv 길이·인코딩 한계 회피 + prompt 수정 시 코드 무수정
- **5게이트 명시 (의지 X 시스템 O)**: AI는 RLHF·확률 분포 영향으로 sycophancy 회귀 위험 상존. 게이트 형식 채워야 끝나는 작업으로 분포 강제 이동. Why: 주인님 지적 "너의 피드백 겉핥기 안 되게" 직접 대응
- **3단계(정보 제공형 훅) 별도 세션 분리**: 셸 선택·sycophancy 키워드 사전 등 별도 디자인 필요. 오늘 무리 안 함
