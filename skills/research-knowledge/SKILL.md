# 지식 리서치 & 문서화 스킬

## 목적
주인님이 요청한 주제를 웹에서 조사하여, 비개발자도 이해할 수 있는 수준으로 문서화한다.
모든 전문 용어는 반드시 설명하고, 예시를 풍부하게 포함하며, 팩트체크를 수행한다.

## 입력
- 사용자가 `/research-knowledge <주제>` 형태로 호출
- 인자가 없으면 주제를 질문

## 실행 프로세스

### 1단계: 리서치 (웹 서칭)
- WebSearch로 최소 2~3개 쿼리를 병렬 실행
- 공식 문서를 최우선, 기술 블로그/커뮤니티는 보조
- WebFetch로 핵심 페이지 2~3개의 상세 내용 수집

### 2단계: 팩트체크
- 공식 문서 vs 블로그/기사 교차 검증
- 수치, 날짜, 버전 등 사실 관계 확인
- 검증 불가 항목은 명시적으로 "미검증" 표기
- 출처 간 충돌 정보 발견 시 양쪽 모두 기록

### 3단계: 문서 작성
- 템플릿: `C:/Users/rlgns/OneDrive/문서/지식/docs/TEMPLATE.md` 참조
- 저장 위치: `C:/Users/rlgns/OneDrive/문서/지식/docs/`
- 파일명: 영문 snake_case (예: `claude_code_channels.md`)

### 문서 작성 규칙

#### 필수 양식 (TEMPLATE.md 기반)
1. **메타정보**: 작성일, 마지막 검증일, 출처 신뢰도
2. **쉬운 설명**: 비개발자 대상, 비유 + 예시 필수
3. **핵심 개념**: 각 개념마다 한 줄 정의 + 비유 + 예시
4. **상세 내용**: 기술적 내용 (전문 용어 첫 등장 시 괄호 설명)
5. **실전 활용**: 구체적 시나리오
6. **제약사항**: 한계, 주의점
7. **팩트체크**: 검증된 사실 / 미검증 / 충돌 정보
8. **용어 사전**: 문서 내 모든 전문 용어 모아서 설명
9. **출처**: URL + 어떤 정보를 참고했는지

#### 문체 규칙
- 비개발자가 읽어도 이해할 수 있는 수준
- 전문 용어가 처음 나올 때마다 반드시 설명
  - 예: "CI(Continuous Integration — 코드를 합칠 때마다 자동으로 테스트하는 시스템)"
  - 예: "MCP(Model Context Protocol — USB 포트처럼 AI와 외부 도구를 연결하는 표준 규격)"
- 비유는 일상적인 것으로 (레스토랑, 우체국, 리모컨 등)
- 코드 예시가 있으면 주석으로 각 줄이 무슨 뜻인지 설명
- 다이어그램은 텍스트 기반으로

### 4단계: 기존 문서 확인
- `docs/index.md`를 먼저 읽어서 같은 주제의 기존 문서가 있는지 확인
- 있으면 새로 만들지 않고 기존 문서를 업데이트
- **기존 파일 업데이트 시 반드시 Edit 도구 사용** — Write(전체 덮어쓰기) 금지. `block-write-docs.sh` 훅에 의해 차단됨
- 새 문서 생성만 Write 허용

### 5단계: 교차 참조 (위키 체계 — rules/wiki.md 준수)
> 문서를 고립시키지 않는다. 관련 문서와 반드시 연결한다.

1. **관련 문서 탐색** — `docs/index.md`에서 관련 카테고리 문서를 찾아 읽기
2. **새 문서에 관련 문서 링크 추가** — 문서 하단에 "관련 문서" 섹션 삽입
   ```markdown
   ---
   ## 관련 문서
   - [문서 제목](파일명.md) — 관련 이유 한 줄
   ```
3. **기존 문서에 역참조 추가** — 관련 기존 문서에도 새 문서 링크를 Edit으로 추가
   - 기존 문서에 "관련 문서" 섹션이 없으면 새로 만들어 추가
4. **index.md 업데이트** — 적절한 카테고리에 새 문서 항목 추가 (Edit으로)
5. **log.md 기록** — `## [YYYY-MM-DD] 동작 | 제목` 형식으로 상단에 추가 (Edit으로)
   - 교차 참조로 업데이트한 기존 문서 목록도 기록

### 6단계: HISTORY.md 업데이트
- 작성/업데이트 이력 기록

## 출력
- 완성된 문서 경로를 주인님에게 보고
- 팩트체크 결과 요약 (검증됨 N건, 미검증 N건, 충돌 N건)
- 교차 참조 결과 (새로 연결한 기존 문서 N건)
